Edge SVG Generator

Using @cloudfour/simple-svg-placeholder and a Netlify Edge Function to generate SVG images on the edge.

Available Options

  • width
  • height
  • text
  • font_family
  • font_weight
  • font_size
  • dy
  • bg_color
  • text_color
  • data_uri
  • charset

See SVG options reference.

Examples

/svg?width=512&height=512
/svg?width=512&height=512&bg_color=000
/svg?width=512&height=512&bg_color=000&text=*
/svg?width=512&height=512&bg_color=000&text_color=FFF

Click (or press Enter on an) image to copy URL to clipboard.
